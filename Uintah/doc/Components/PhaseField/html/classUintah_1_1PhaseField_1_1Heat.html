Detailed Description

template<VarType VAR, DimType DIM, StnType STN, bool AMR = false>
class Uintah::PhaseField::Heat< VAR, DIM, STN, AMR >

Heat PhaseField applications.

Implements a Finite Difference solver for the simulation of the heat diffusion model

with initial data

where ranges over the problem dimension.

The model parameters are:

  • thermal diffusivity
