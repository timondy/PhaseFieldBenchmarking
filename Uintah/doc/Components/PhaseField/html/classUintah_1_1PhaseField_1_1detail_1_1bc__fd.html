Detailed Description

template<typename Field, StnType STN, VarType VAR, Patch::FaceType F, BC B, int GN, FC C2F = FC::None, bool CNEW = false>
class Uintah::PhaseField::detail::bc_fd< Field, STN, VAR, F, B, GN, C2F, CNEW >

Abstract finite-differences scheme for variables at grid boundary.

To implement a new boundary-condition it should be sufficient to define a new BC and code the relevant implementations of this class (for amr simulations also the corresponding bc_fd implementation for fine/coarse interfaces must be provided)
