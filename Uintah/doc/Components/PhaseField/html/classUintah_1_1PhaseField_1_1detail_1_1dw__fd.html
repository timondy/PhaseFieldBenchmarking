Detailed Description

template<typename Field, StnType STN, VarType VAR, int GN>
class Uintah::PhaseField::detail::dw_fd< Field, STN, VAR, GN >

Abstract finite-differences scheme for dw variables.

To implement a new finite-differences scheme it should be sufficient to define a new STN and code the relevant implementations of this class (for amr simulations also the corresponding bc_fd implementation for fine/coarse interfaces must be provided)
