Detailed Description

template<typename T, typename Problem, size_t... I>
class Uintah::PhaseField::detail::amr_interpolator< ScalarField< T >, Problem, index_sequence< I... >, I1, D1 >

Wrapper of grid variables for interpolation from coarser to finer levels (1D linear implementation)

Implements linear interpolation of a variable from coarser to finer levels in 1D

Template Parameters
Tvariable data type (must be constant)
Problemtype of PhaseField problem
Ilist of indices corresponding to the variable within the subproblems
Todo:
generalize implementation to arbitrary dimension
