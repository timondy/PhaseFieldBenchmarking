Detailed Description

template<typename T, StnType STN, VarType VAR>
class Uintah::PhaseField::detail::dw_fd< ScalarField< T >, STN, VAR, 1 >

Finite-differences scheme for dw variables (1-ghost in each direction) (ScalarField implementation)

Implement P3 (three-point 1D), P5 (five-point 2D) and P7 (seven-point 3D) stencils

Template Parameters
Ttype of the field value at each point
STNfinite-difference stencil
VARtype of variable representation
