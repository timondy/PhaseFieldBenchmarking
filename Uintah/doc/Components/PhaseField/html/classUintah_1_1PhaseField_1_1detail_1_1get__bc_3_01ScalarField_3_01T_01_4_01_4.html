Detailed Description

template<typename T>
class Uintah::PhaseField::detail::get_bc< ScalarField< T > >

Get boundary conditions on one face static functor (ScalarField implementation)

Allows to get the boundary value for a given variable label on a given boundary face (face, child) of the given patch.

Remarks
Refactor of getBCValue function in Core/Grid/BoundaryConditions/BCUtils.h
