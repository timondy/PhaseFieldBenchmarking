Detailed Description

template<VarType VAR, StnType STN, typename... Field>
class Uintah::PhaseField::Problem< VAR, STN, Field >

PhaseField Problem.

It represent one portion of the global problem on which all implementations required to solve a timestep are fixed (i.e: there is no need to check boundary conditions/variables extents/neighbors...)

Template Parameters
VARtype of variable representation
STNfinite-difference stencil
Fieldlist of type of fields (ScalarField < T > or VectorialField < T, N >)
