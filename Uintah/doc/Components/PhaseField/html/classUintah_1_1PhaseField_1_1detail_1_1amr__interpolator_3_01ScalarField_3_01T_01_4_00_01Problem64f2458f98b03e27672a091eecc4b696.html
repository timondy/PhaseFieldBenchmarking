Detailed Description

template<typename T, typename Problem, size_t... I, DimType DIM>
class Uintah::PhaseField::detail::amr_interpolator< ScalarField< T >, Problem, index_sequence< I... >, I0, DIM >

Wrapper of grid variables for interpolation from coarser to finer levels (piecewise implementation)

implements piecewise constant interpolation of a variable from coarser to finer levels in 1D

Template Parameters
Tvariable data type (must be constant)
Problemtype of PhaseField problem
Ilist of indices corresponding to the variable within the subproblems
DIMproblem dimension
