template<typename T, T... I>
struct Uintah::PhaseField::make_integer_sequence< T, 0, I... >

A helper alias template make_integer_sequence is defined to simplify creation of integer_sequence types with 0, 1, 2, ..., N-1 as I. (N=0 implementation)
