Detailed Description

template<typename T, StnType STN, typename Problem, typename Index, BCF ... P>
class Uintah::PhaseField::detail::bcfd_view< ScalarField< T >, STN, Problem, Index, P... >

Wrapper of DataWarehouse variables for both basic and complex differential operations over both physical and amr boundaries (ScalarField implementation)

Adds to dw_view the possibility to compute finite-difference approximation of of differential operations at boundary cells/points

Template Parameters
Problemtype of PhaseField problem
Indexindex_sequence of Field within Problem (first element is variable index, following ones, if present, are the component index within the variable)
Plist of BC, FC, and Patch::Face packs
