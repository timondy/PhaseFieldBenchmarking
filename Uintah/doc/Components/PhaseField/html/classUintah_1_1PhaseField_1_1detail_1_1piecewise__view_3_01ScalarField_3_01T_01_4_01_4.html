Detailed Description

template<typename T>
class Uintah::PhaseField::detail::piecewise_view< ScalarField< T > >

Container for collecting multiple views as a unique composite view (ScalarField implementation)

It is used as abase class for views that are defined across multiple sub-problems (i.e. bcs views and amr coarser and finer views)
