Detailed Description

template<VarType VAR, DimType DIM, StnType STN, bool AMR = false>
class Uintah::PhaseField::PureMetal< VAR, DIM, STN, AMR >

PureMetal PhaseField applications.

Implements a Finite Difference solver for the simulation of the anisotropic solidification of an under-cooled metal around a solid seed using the model from

A, Karma and W.-J. Rappel, "Phase-Field Method for Computationally Efficient Modeling of Solidification with Arbitrary Interface Kinetics", Phys.Rev.E, 1996.

The model parameters are:

  • coupling parameter
  • width scaling ( is the interface width)
  • time scaling ( is the characteristic time of attachment of atoms at the interface)
  • anisotropy strength
  • thermal diffusivity

The following no-dimensionalizations are performed
