template<typename T, size_t N>
class Uintah::PhaseField::detail::get_bc< VectorField< T, N > >

Get boundary conditions on one face static functor (VectorField implementation)

Allows to get the boundary value for a given variable label on a given boundary face (face, child) of the given patch.
