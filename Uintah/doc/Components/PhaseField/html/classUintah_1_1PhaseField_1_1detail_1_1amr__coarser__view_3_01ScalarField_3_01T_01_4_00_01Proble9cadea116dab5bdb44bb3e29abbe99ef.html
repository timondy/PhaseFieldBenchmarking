template<typename T, typename Problem, size_t... I>
class Uintah::PhaseField::detail::amr_coarser_view< ScalarField< T >, Problem, index_sequence< I... > >

Composite view for accessing coarser grid variables (ScalarField implementation)

Remarks
deletion of views instantiated in piecewise_view are performed here
Template Parameters
Ttype of the field value at each point
Problemtype of PhaseField problem
Ilist of indices of Field within Problem (first element is variable index, following ones, if present, are the component index within the variable)
