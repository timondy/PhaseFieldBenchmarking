Detailed Description

template<typename T, StnType STN>
class Uintah::PhaseField::detail::basic_fd_view< ScalarField< T >, STN >

Abstract class for basic finite-difference operations on variables (ScalarField implementation)

detail implementation of view (variable wrapping) which include also finite difference approximation on basic differential operations (first and second oreder derivatives)

Template Parameters
Ttype of the field value at each point
STNfinite-difference stencil
